**1. ¿Qué hace @RequestBody?**
El @RequestBody Recibe la petición de http y la convierte a objeto java

**2. ¿Para qué sirve @Valid?**
@Valid valida el objeto que recibe con etiquetas como@NotNull, @NotBlank.

**3. Diferencia entre @NotNull y @NotBlank**
@NotNull no recibe objetos null y @NotBlank no permite espacios, ni null, ni vacíos.

**4. ¿Por qué usamos List en vez de arreglo?**
Usamos List en vez de arreglos porque los arreglos tienen un tamaño fijo inmutable y List sirve para crecer dinámicamente, además de los métodos que permite usar.
